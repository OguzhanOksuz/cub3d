
# Cub3d

Type ```make run``` to run it. \
works in MacOs

## Input file

*.cub \
path for textures \
floor and ceiling colors \
map

NO ./assests/textures/KU.xpm
SO ./assests/textures/GU.xpm
EA ./assests/textures/DO.xpm
WE ./assests/textures/BA.xpm

F 120,80,10
